# Adoback 更新计划

## 第一梯队 — 核心闭环

### 1. 备份回滚/恢复 `adoback restore`
- [ ] 交互式选择：列出某个文件的所有历史快照
- [ ] 支持按文件名模糊搜索
- [ ] 选择快照后恢复到原路径（自动备份当前版本再覆盖）
- [ ] `--list` 仅查看快照列表不恢复
- **目标版本：v0.5.0**

### 2. 备份空间管理 `adoback clean`
- [ ] 保留最近 N 个快照（默认 10）
- [ ] 超过 X 天自动清理（默认 30 天）
- [ ] 总容量上限（默认 10GB）
- [ ] `--dry-run` 预览将清理的文件
- [ ] `--force` 跳过确认直接清理
- [ ] 守护模式下自动执行清理策略
- **目标版本：v0.5.0**

### 3. 桌面通知
- [ ] macOS 原生通知（osascript / terminal-notifier）
- [ ] 备份完成 / 失败 / 磁盘空间不足时触发
- [ ] 配置项：可关闭通知
- **目标版本：v0.6.0**

---

## 第二梯队 — 体验提升

### 4. TUI 实时仪表盘 `adoback status`
- [ ] 终端实时刷新面板
- [ ] 显示：上次备份时间、监控目录状态、磁盘占用、最近日志
- [ ] 纯 ANSI 实现，零依赖
- **目标版本：v0.6.0**

### 5. 文件变更监听模式（fsevents）
- [ ] 替代定时轮询，使用 macOS 原生 fsevents
- [ ] 文件保存瞬间触发备份
- [ ] 降低 CPU 占用
- **目标版本：v0.7.0**

### 6. 忽略规则 `backup.ignore`
- [ ] 类似 .gitignore 的排除模式
- [ ] 默认排除 Adobe 缓存、Media Cache、临时文件
- [ ] 配置文件中 `backup.ignore_patterns = [...]`
- **目标版本：v0.6.0**

---

## 第三梯队 — 锦上添花

### 7. 云端/外置同步（可选）
- [ ] 支持备份到 iCloud Drive / 外置硬盘 / NAS
- **目标版本：v0.8.0+**

### 8. 备份时间线可视化
- [ ] 文件大小变化趋势
- [ ] 备份历史时间线
- **目标版本：v0.8.0+**

### 9. Web UI
- [ ] 本地 HTTP 服务，浏览器管理备份
- [ ] 面向不习惯终端的设计师用户
- **目标版本：v1.0.0+**
